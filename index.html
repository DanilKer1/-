GIT Flow

GITclone - для того, чтобы "склонить" репозиторий

GITpull - для того, чтобы получить изменения с репозитория

GITcheckout -b branchname - создать новую ветку и переключить на неё

GITadd - добавить изменения, которые сделали локально

GITcommit -m "commit text" - подписать их

GITpush orign branchname - запушить git сервер
